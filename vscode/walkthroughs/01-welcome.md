# SuperBased Observer

**One local intelligence layer for every AI coding tool you use.**

Observer captures, normalises, and analyses tool-call activity from
Claude Code, Codex, Cursor, Cline, Copilot, Roo Code, Continue, and
several others — all locally. No telemetry. No data leaves your
machine.

This walkthrough takes about **5 minutes** and covers:

- Verifying the binary is installed and healthy
- Picking how the extension manages the observer daemon
- Opening the dashboard inside VS Code
- Routing your AI CLIs through the observer proxy so token counts
  are accurate
- Auto-updating your `CLAUDE.md` / `AGENTS.md` / `.cursorrules`
  from the patterns Observer learns about your workflow

You can revisit any step later via **Help → Welcome** or the
**Get Started: Open Walkthrough…** command.
